### Git & Github 요약

git: 코드의 변경 기록을 관리하는 프로그램</br>
누가 언제 무엇을 수정했는지 기록, 여러 사람이 협업할 때 충돌 관리</br>

github: git 기록을 공유하는 프로그램</br>
여러 사람이 코드를 올리고 합치는 공간</br>

기본 흐름: git add(수정파일 추적)-> git commit(기록 저장) -> git push(github에 업로드)</br>

중요 : git pull을 꼭 해야 충돌이 발생하지 않음 !! </br>

### Django (장고 요약)
python 기반 소프트웨어 개발을 효율적으로 만들어주는 도구</br>
흐름 : Django 프로젝트 생성-> 가상환경 설정 -> git init-> .gitignore 설정 -> github 연결 -> commit and push </br>

1. project : 웹사이트 전체를 의미 </br>
2. app : 실제 기능 구현을 위한 코드들 </br>
