# remote : github 사이트랑 연결해주는 느낌

# commit : 파일 옆에 부연 설명 써주는 거 같음
# push : 내가 변경한거 올리기
# pull : 프로젝트 받아오기
# origin : 이건 내 개인의 것
# upstream : 이건 공동의 것
# main (branch) : 말 그대로 가지? 마인드맵으로 따지면 작은가지 느낌임
